# 🎥 Real-Time Interview Platform

A full-stack MERN application for conducting technical interviews with **real-time video communication, shared code editor, and AI-powered evaluation system**.

---

## 🚀 Live Demo

👉https://interview-platform-delta-gilt.vercel.app

---

## ⚡ Key Features

* 🔐 **Authentication** – Secure login/signup with JWT
* 🎥 **WebRTC Video Calls** – Real-time peer-to-peer communication
* 💻 **Live Code Editor** – Monaco Editor with real-time sync
* ⏱️ **Interview Timer** – Synchronized across users
* ⭐ **Candidate Rating System** – Feedback + scoring
* 📊 **Interview History** – Track past sessions
* 📡 **Real-time Collaboration** – Powered by Socket.io

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/DiwakarNallappagari/Interview-platform.git
cd interview-platform
```

### 2. Install Dependencies

```bash
cd backend
npm install

cd ../frontend
npm install
```

### 3. Run Project

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```
